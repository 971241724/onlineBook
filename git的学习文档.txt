
					Git的学习总结
一、

	1.使用cd指令指定路径(根目录是"/")
	2.pwd指令查看当前的目录
	3.git init指令在当前目录下创建本地仓库
	4.git add指令添加文件到仓库，git commit指令提交文件到仓库
	5.git log查看log

	git config --global user.name "gaohuan"
